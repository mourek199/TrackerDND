# Tracker DND 
## Overview 🧾
- This is a tracker app designed to be used as
a background for famous board game Dungeons & Dragons

## Features 🎮
- Current location setting and situation description panel
- Present minifigures colors description
- Queue used for keeping track on whose turn it is
- Creature conditions (Blinded, Charmed, Grappled and more)
- Quick remove buttons for specified Categories

## Technologies 🔧
- Java (openjdk-24)
- Json ()
- Java Swing
